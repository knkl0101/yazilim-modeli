# yazilim-modeli (C.O.R.A)

Next.js ile geliştirilen **C.O.R.A** sürekli test döngüsü web platformu.

## Geliştirme

```bash
npm install
npm run dev
```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini açın (port doluysa terminalde gösterilen adresi kullanın).

## Build

```bash
npm run build
npm run start
```

## Teknoloji

Bu proje [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) ile oluşturulmuştur. Ayrıntılar için [Next.js belgeleri](https://nextjs.org/docs).
